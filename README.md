# VerseLang Required Submission

This folder contains the required implementation and five demonstration programs for VerseLang.

## Files

- `verselang.py` — interpreter
- `hello_world.verse` — simple program 1
- `math_demo.verse` — simple program 2
- `conditions_demo.verse` — simple program 3
- `messaging_demo.verse` — complex program
- `fizzbuzz.verse` — FizzBuzz program

## Run Commands

```bash
python verselang.py hello_world.verse
python verselang.py math_demo.verse
python verselang.py conditions_demo.verse
python verselang.py messaging_demo.verse
python verselang.py fizzbuzz.verse
```

## Notes

Keep every `let ... taste ...` assignment on one line.
